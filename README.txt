Organización de las carpetas:

	* Aplicaciones Android:
		- app_guia_*: distintas versiones de BlindBit.
			app_guia_v5 es la última versión.
		- cuadrantes_v1: indica la distancia estimada a la que
			se encuentran los beacons.
		- miniapp: indicia en qué rango (INMEDIATE, NEAR,
			FAR, UNKNOWN) están los 3 beacons escaneados.
		- miniappDistace: como la anterior pero perminte saber
			la distancia estimada.
		- pruebaSonido: indica, mediante la variación de la 
			frecuencia de emisión de un sonido, cómo de cerca
			se encuentra el beacon más cercano.
		-TomcatClient: permite establecer la conexión con el 
			servidor por medio de webSockets.
			
	* Código Java (Eclipse):
		- Eclipse_ws: contiene las distintas versiones del servidor.
		ServidorTomcat_cliente_v5 es la última versión.
	
	* Memoria:
		- Memoria: contiene el código latex de toda la memoria.

	* XML:
		- XML edificio: contiene todos los archivos XML
			desarrollados. Los finales están en xml_modif
			para la Facultad y xml_modif_belen para los de
			la vivienda.
